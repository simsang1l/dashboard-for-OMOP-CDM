## 3. 구현할 사항
**당뇨관련 대시보드 구현**
  > - 연도별 환자 수, 방문(남녀, 전체)
  > - follow up 기간 별 환자 현황
  > - 환자별 처방 약물(원 그래프)
  > - 약물별 비용 정보
  > - 코드 매핑률, 데이터 매핑률
  > - 테이블별 데이터 커버리지
  > - 당뇨 환자의 상병 리스트
  > - 검사 변화 정도
